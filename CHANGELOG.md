- Added `bin` configuration to enable `npx waar` command for easy execution
- Added shebang line to index.ts for proper executable support

### Changed
- Updated README.md to show `npx waar` as the primary usage method
- Translated default auto-reply message from Spanish to English (more universal)

### Fixed
- Package can now be executed directly via npx without cloning the repository
